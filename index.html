The Color System

Blue — Structure

Things that create. def class lambda import from. Blue suggests blueprints, architecture, the skeleton of your program.

Amber — Flow

Things that direct. if for while return try raise. Amber means caution — execution could go different ways here.

Green — Values

Things that are. "strings" 42 3.14 True None. Green is ground truth, the concrete data your program operates on.

Purple — Logic

Things that reason. and or not is in ==. Purple marks the decision-making layer of your code.

Cyan — References

Things that point. func() print() len(). Cyan sits between blue (structure) and green (values) — references use structure to access values.

Gray — Meta

Things that describe. # comments type hints @decorators self. Not executed, not competing — informational elements fade.
